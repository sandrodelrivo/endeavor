# Endeavour

A heavily-modded Minecraft 1.21.1 NeoForge server for ~6 friends. The world is disconnected, dangerous, and dead. Players cooperate to rebuild civilization in a world where a prior age left ruins, dungeons, and infrastructure but no people.

## What's in this repo

- `design/` — the full design document, tier map spreadsheet, mod list, research notes, open questions
- `datapack-worldgen/` — `zzz_endeavour_worldgen` datapack: climate gradient, biome-locked ores, forced nether portals
- `datapack-rules/` — `zzz_endeavour_rules` datapack: sprint disable, respawn disable, mob scaling
- `mod/` — `endeavour` companion mod: Patchouli book, advancements, lore items, ignition rules
- `textures-source/` — hand-made ore textures pending migration into the mod
